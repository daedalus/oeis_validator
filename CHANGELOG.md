## [0.1.0] - 2026-05-11

### Added
- Initial release as installable Python package (hatchling build backend)
- Validate OEIS internal-format `.txt` draft entries
- 90+ validation rules across all 19 field tags
- CLI with file, stdin, demo, and coverage modes
- Full pytest suite with 80%+ coverage target
- Ruff format + prospector (ruff check, mypy, pylint) + semgrep + vulture CI
- `docs/SPEC.md` specification-first contract
- AGENTS.md for agentic coding workflows
